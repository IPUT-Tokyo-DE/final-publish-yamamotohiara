# ここにゲームのタイトルを記述する
motimoti
## ゲームの内容
キャラクターを操作して、上から落ちてくるおもちを集めるゲームです。
落下物は、おもちか障害物のどちらかです。
障害物を避けながら、おもちを集めてください。
ゲーム終了時に、最も多くのおもちを集めたプレイヤーが勝利となります。

![画面イメージ](docs/images/game_image01.png)

## 操作説明
左スティックでキャラクターを移動します。
